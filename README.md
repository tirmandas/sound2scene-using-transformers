# sound2scene-using-transformers
create a directory name it sound2scene.
change directory to sound2scene(cd sound2scene)
update all files in the directory except earthquake.wav,tsunami.wav,fire.wav.
create a folder inside sound2scene project structure and name it as "sounds".
update earthquake.wav,tsunami.wav,fire.wav in sounds folder.
install blender
after updating all files run python main.py "any sound name" in terminal .
ex:python main.py tsunami
